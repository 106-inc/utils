## csv_to_tex
Converter from table in ```CSV``` format to ```tex``` code.
Usage:
* Change script file rights:
  ```bash
  $ sudo chmod +x csv_to_tex.py
  ```
* Run:
  ```bash
  $ ./csv_to_tex.py source_file.csv dest_file.tex
  ```
